# 📚 Library Management System

An interactive command-line application built with **Python** for managing library books and loan workflows, featuring data persistence using **JSON**.

---

## 🌟 Key Features

* 📖 **Book Management:** Full CRUD operations (Create, Read, Update, Delete) for the library catalog.
* 🔄 **Data Persistence:** Records are dynamically loaded and saved using native **JSON** format.
* 🛡️ **Error Handling & Input Validation:** Intercepts invalid CLI inputs to prevent system crashes and ensure smooth user experience.
* 🔍 **Search & Filter:** Easily filter books by availability, author, or category.

---

## 🛠️ Tech Stack

* **Language:** Python 3.x
* **Data Format:** JSON
* **Version Control:** Git & GitHub
